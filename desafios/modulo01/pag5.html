Os super links

Links é a essência fundamental do hypertexto.Permite ligar documentos entre si e permite a navegação entre essas páginas.

Para criar um hyperlink devemos criar âncoras através da tag <a>

Controlando o destino: Sou rei

Para isso vamos usar o atributo target que tem os seguintes valores

_blank - nova janela

_self - mesma janela (padrão)

Os outros não importa, é isso, foda-se

Outro recurso interessante é o rel que indica qual é natureza do destino e aceita valores como:

next - link é para a próxima página

prev - link é para a página anterior

author - link é para oo site do autor do artigo atual

external - link é para outro site que não faz parte do site

nofollow - link pra um site não endossado, como um link pago

Download

Basta fazer o link direto para o arquivo e adicionar o atributo <download> e o atributo type para indicar o tipo de arquivo 


<a href="arquivo/meuLivro.pdf" <download>="meuLivro.pdf" type="aplication/pdf"><a>
