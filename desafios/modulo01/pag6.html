Imagens dinâmicas, áudio e vídeos

Fotos, vídeos e áudios são essências para a construção de sites hoje em dia .Ilustrar bem o seu conteúdo é imprescíndivel para deixar claro a mensagem que queremos passar.

Imagens flexíveis

Primeiro precisamos gerar imagens de tamanhos diferentes e a fazer o navegador carregar a imagem certa para cada situação. para isso use a tag <picture>.

Código base


<picture>
    <img src="foto-g.png" alt="imagem flexível">
</picture>

O problema é quando a imagem chega em 1000px, ela não vai caber na tela, para resolver isso usamos a tag <source> 


<picture>
    <source midia="(max-width: 1050px)" srcset="foto-m.png" type="image/png">
    <img src="foto-g.png" alt="imagem flexível">
<picture>

A tag source possui três atributos:

  1. type: indica a media type da imagem(tipo)
  2. srcset: nome da imagem que será carregada quando o limite for atingido
  3. midia: indica o tamanho máximo a ser considerado para carregar a imagem do srcset

Importante:Existe uma ordem no source, começa sempre do menor e o último item é a imagem padrão.

Áudios

Do mais simples possivel, tag <audio>


<audio preload="metadata" controls autoplay loop>
    <source src="midia/guanacast-3.mp3" type="audio/mpeg">
    <source src="midia/guanacast-3.ogg" type="audio/ogg">
    <p>Não suporta áudio <a href="midia/"guanacast-3.mp3" download="guanacast-3.mp3" type="audio/mpeg"
    <a><p>
</audio>

Atributos da tag audio

  • preload - O áudio será pré-carregado ou não?
    • metadata - Apenas as informações do arquivo (tamanho, tempo, informações de direitos...
    • none - Não carrega nada até o botão player ser clicado
    • auto (padrão) - carrega o arquivo de áudio inteiro assim que a página for carregada
  • Controls - apresenta o player na tela
  • autoplay - inicia a reprodução de áudio assim que a página for carregada
  • loop - È um loop

Vídeos

Usando a tag <video> caso o vídeio esteja hospedado no próprio servidor, não é muito recomendado


<video width="600" poster="thumn.jpg" controls>
    <source src="meu-video.mp4" type="video/mp4">
    <source src="meu-video.webm" type="video/webm">
<video>

vamos para os atributos

  • width - largura do vídeio
  • poster - capa
  • controls - apresenta o player

Esse é um vídeio hospedado no youtube, melhor que no próprio site
