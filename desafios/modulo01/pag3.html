Hierarquia de Títulos

Títulos são conhecidos como headings e possuem seis niveis de hierarquia

<h1>Títuo principal

<h2>Sub-assunto de h1

<h3>Sub-assunto de h2 e assim por diante até h6

Formatação de textos

Semântica: é o significado dos vocábulos, oposição à sua forma

HTML5 é focado na semântica, ou seja, as tags tem significado, por exemplo:

Negrito e Itálico

tag <strong> - o termo delimitado possui força dentro da frase. negrito

tag <em>- significa dar ênfase ao termo.itálico

Marca texto

Tag <mark> - serve para marcar uma parte do texto

Small

tag <small> - deixa o texto menor

Texto deletado

Tag <del> - o texto deve ser desconsiderado pelo leitor

Texto Inserido

Tag <ins>- sublinha o texto e indica que deve prestar atenção nele

Textos sobrescrito e subscrito

tag <sub> - x2

tag <sup> - x2

Trechos de códigos

Tag <code> - indica para o navegador que se trata de um código

è preciso usa-la junto com a tag <pre> para um visual interessante


for n in range(1, 10):
    print(n)

Citações

Podemos usar a tag <q>, que não tera nenhum deslocamento. È mais
usada quando queremos uma citação no meio do parágrafo

Também tem para citações mais longas (bloco) e que tenham um parágrafo
só para si. Colocamos tudo dentro de <blockquote>. Dá pra colocar um
link para o texto original, usando o parâmetro cite dentro da tag

Abreviação

tag <abbr> mostra o significado da sigla quando passa o mouse


<p>Estudando <abbr title="Hyper Text Markup Language">
HTML</abbr>para criar sites.<p>

A tag mais inútil

Tag <bdo> - inverte o texto, tem parâmetro dir.

rtl = direita para a esquerda (right-to-left)

ltr = esquerda para a direita (left-to-rigth)
